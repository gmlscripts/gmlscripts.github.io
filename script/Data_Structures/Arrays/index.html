Data Structures :: Arrays

array(value,...)
Returns an array populated by given arguments.
array_add(array,value,...)
Returns the given array with elements appended to it.
array_sort(array, ascend)
Returns a sorted version of a given array.
explode(delimiter,string)
Returns an array of strings parsed from a given string of elements separated by a delimiter.
explode_real(delimiter,string)
Returns an array of real values parsed from a given string of elements separated by a delimiter.
implode(delimiter,array)
Returns a string of elements from a given array of strings and separated by a delimiter.
implode_real(delimiter,array)
Returns a string of elements from a given array of real values and separated by a delimiter.
